Spovio is an HR management system covering the full employee lifecycle — time-off,
attendance, payroll and document storage. The frontend is a React application; the
backend is Django, with the time-sensitive work (accrual, scheduled jobs) handled
by Celery workers over a Redis broker so the request path stays responsive.

The linked case study goes deeper into the role-based accrual model, the monthly
payroll generation and export pipeline, the AWS S3 storage design, and how email
and SMS notifications drive the approval workflows that automated most of the
manual HR process.
