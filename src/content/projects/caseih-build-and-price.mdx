CaseIH Build & Price is a configurator for agricultural machinery, built at
Coolshop from scratch in React 18. Rather than a single-market SPA, it was
designed as a Storybook-documented component library: regions differ by theme
and catalogue, not by codebase. Styled Components carries per-region branding,
GraphQL/Apollo provides the typed data layer, and server-side rendering keeps a
heavy, image-rich product surface fast and indexable.

The same components compose into 5+ region-specific deployments, with Jest unit
tests and GitHub CI/CD keeping the library releasable as it grows. The linked
case study walks through the component-library and theming model, the GraphQL
data layer, the SSR setup for SEO, and how reuse across regions was kept
maintainable.
