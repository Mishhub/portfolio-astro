Zenit World is a copy-trading platform serving 3M+ users, where followers mirror
lead traders' strategies onto their own wallets. The backend is a modular Django
4.1 + PostgreSQL service: custody and transaction orchestration are delegated to
**Fireblocks**, onboarding runs through **SumSub** KYC/AML in real time, and a
custom **ZENIT** token is exposed with both fiat and crypto wallet support.

Staking, yield farming and the automated copy-trading algorithms are driven by
smart-contract integrations, while **Redis + Celery** handle audit logging,
event tracking and the background work the request path can't block on. User
documents and assets live in **MinIO** behind S3-compatible storage.

The linked case study walks through the custody model, the real-time KYC/AML
flow, the copy-trading and token design, and how audit logging and event
tracking stay durable without slowing the request path.
